# Guía de IA para Productividad

Guía práctica para usar Claude.ai (y comparar el resto de herramientas) en el trabajo diario. **No se necesitan conocimientos técnicos.**

---

## ¿Para quién es esta guía?

Para cualquier perfil que quiera ir más allá de usar la IA como buscador:

- 🧑‍💼 **Preventa y consultoría** — generar propuestas, preparar demos, crear documentación de calidad rápidamente
- 🧑‍🔧 **Solutioning técnico** — redactar arquitecturas, generar diagramas, documentar decisiones de diseño
- 🧑‍💼 **Recruiting y RRHH** — analizar CVs, generar informes de candidatos, estructurar entrevistas
- 📋 **Management y operaciones** — resumir reuniones, crear planes de proyecto, analizar datos en Excel

> No necesitas saber programar. Todo funciona desde el navegador.

---

## Qué vas a conseguir

Al terminar esta guía serás capaz de:

- Elegir la herramienta adecuada para cada tarea (ChatGPT, Gemini, Copilot 365, Claude.ai)
- Escribir prompts que obtengan resultados útiles desde el primer intento
- Configurar Claude.ai para que trabaje según tu rol y contexto habitual
- Generar documentos Word, Excel y PowerPoint con formato real, sin copiar y pegar
- Conectar Claude.ai a Jira, Google Drive, Azure DevOps u otros servicios
- Montar un agente de equipo (Project) con conocimiento y configuración compartidos
